# 데이터 시각화

데이터 시각화를 배웠다. 보편적으로 알고 있던 선차트 바차트 외에도 파이차트 히스토그램 산점도 등의 다양한 플롯을 배워서 어렵기도 했고 새롭기도 했다.

```
수치형 데이터 : 사칙 연산이 가능한 데이터
    연속형 데이터 : 값이 끊기지 않고 연속된 데이터 float
    이산형 데이터 : 정수로 딱 떨어져 셀 수 있는 데이터 int
      초밥 : 12g (연속형 데이터) / 밥알 수 320, 280개? 280.5개. (이산형)
      
      수치형 데이터 시각화
      sns.kdeplot() 커널밀도추정 함수 그래프
      sns.displot() 분포도
      sns.rugplot() 러그플롯
      
      
범주형 데이터 : 범주로 나누어지는 데이터
    순서형 데이터 : 순위 ranking 을 매길 수 있는 데이터 (비교가 가능하다) / 크고 작음 
      학점 (숫자로 나오는게 아니라, 분류체계), 스포츠 경기 순위
    명목형 데이터 : 순위가 따로 없는, 비교할 수 없는, 더 높고 낮고 X 
      데이터 성별, 우편번호, 반, 소속들
      
      
      범주형 데이터 시각화
      sns.barplot() 막대 그래프
      sns.pointplot() 포인트플롯
      sns.boxplot() 박스 플롯 
      sns.violinplot() 바이올린 플롯 
      sns.countplot() 카운트 플롯
      plt.pie() 파이 그래프 seaborn에 없고 matplotlib에만 있음
